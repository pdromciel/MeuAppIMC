# 📱 MeuApp IMC

Este é um projeto desenvolvido para fins acadêmicos na disciplina de programação, com o objetivo de praticar o desenvolvimento de aplicações móveis utilizando React Native / Expo.

---

## 📌 Sobre o projeto

O **MeuApp IMC** é um aplicativo simples que realiza o cálculo do Índice de Massa Corporal (IMC).  
A partir do peso e altura informados pelo usuário, o sistema retorna o valor do IMC e sua classificação.

---

## 🎯 Objetivo

- Praticar desenvolvimento mobile
- Aplicar conceitos de lógica de programação
- Trabalhar com entrada de dados do usuário
- Implementar cálculos matemáticos simples
- Desenvolver interface básica em aplicativo

---

## 🧮 Funcionalidades

- Inserção de peso (kg)
- Inserção de altura (m)
- Cálculo automático do IMC
- Exibição do resultado com classificação (baixo peso, normal, sobrepeso, etc.)

---

## 🛠️ Tecnologias utilizadas

- React Native
- Expo
- JavaScript / TypeScript
- Node.js

---
